InheritanceTilePool Object

This post http://lab.polygonal.de/2008/06/18/using-object-pools/ suggests that using Object pools, especially for complex classes like Sprite is a lot faster than calling new Object(). The suggested implementation uses a linked list, but to get started with it here I'm using an Array. If anyone wants to try it with a linked list and compare the times, it seems like it could be worth it :)
