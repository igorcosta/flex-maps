GeoMap()Constructor
public function GeoMap(width:Number = 320, height:Number = 240, draggable:Boolean = true, mapProvider:IMapProvider = null, ... rest)

Initialize the map: set properties, add a tile grid, draw it. Default extent covers the entire globe, (+/-85, +/-180).

Parameters
width:Number (default = 320) — of map, in pixels.
 
height:Number (default = 240) — of map, in pixels.
 
draggable:Boolean (default = true) — the map can be dragged or not.
 
mapProvider:IMapProvider (default = null) — map provider, e.g. Blue Marble.
 
... rest — a MapExtent or a Location and zoom (comma separated)

See also

public function setExtent(extent:MapExtent):void
